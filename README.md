# Funpay scraper API
API for scraping information about users and their lots on the FunPay website.

## Getting started
1. Creave venv
```bash
python -m venv venv
python3 -m venv venv
```

2. Select interpreter
```bash
Left Ctrl + Left Shift + P
> Select Interpreter
```

## Usefull commands
1. Create/Update requirements.txt
```bash
pip freeze > requirements.txt
```
